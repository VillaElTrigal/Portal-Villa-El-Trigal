LISTO PARA GITHUB PAGES

1. Cree un repositorio público en GitHub, por ejemplo: portal-villa-el-trigal
2. Suba TODO el contenido de esta carpeta a la raíz del repositorio.
3. En GitHub abra Settings > Pages.
4. En Build and deployment seleccione: Deploy from a branch.
5. Elija la rama main y la carpeta / (root), luego presione Save.
6. Espere unos minutos. GitHub mostrará el enlace publicado.

PARA ACTUALIZAR DESPUÉS
Reemplace o suba los archivos modificados al mismo repositorio. GitHub Pages publicará los cambios automáticamente.
