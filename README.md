# SIGVE — Junta de Vecinos Villa El Trigal

**Versión:** 1.1.0 — Gestión Financiera Integrada

Portal comunitario y panel administrativo desarrollado para la gestión real de la Junta de Vecinos Villa El Trigal.

## Instalación de esta versión

Ejecutar `ACTUALIZAR_SUPABASE_SIGVE_V1_1_0.sql` en Supabase y luego publicar el contenido completo en GitHub Pages. Las instrucciones detalladas están en `LEEME_SIGVE_V1_1_0.txt`.

## Componentes principales

- Gestión de socios y niños del hogar.
- Reservas de sede y Zumba.
- Cuotas mensuales.
- Certificados de residencia.
- Finanzas y Libro de Caja.
- Informe financiero mensual.
- Portal público y administración de contenidos.
